# はじめに

AWS認定に合格すると、模擬試験を無料で受験できるバウチャーが発行される。「[WEB問題集で学習しよう](https://aws.koiwaclub.com/)」の専門分野は問題数が少ないので模擬試験を受けようしたら、申込みで手間取った話。

# 要約

* バウチャーを利用して模擬試験を受ける場合は**PSI**で予約する

# AWS認定の模擬試験でバウチャーを利用できない

## ピアソンVUEで申し込む

AWS認定試験をオンラインで受験する場合、PSIとピアソンVUEのどちらかを選択できる。日本語に対応しているピアソンVUEを利用していたので、模擬試験もピアソンVUEで申し込もうとサイトにアクセス。いつもと同じように受験する科目をポチポチ選択し、入力画面でバウチャーを適用すると以下が表示される。

![バウチャー無効](https://storage.googleapis.com/zenn-user-upload/f405f6cf590268206dc62740.png)

あれ、入力するバウチャー間違えたかな？と思い、別のバウチャーを入力するも結果は同じ。

https://twitter.com/sunazukin_it/status/1426882854567383042

## AWSに問い合わせる

[よくある質問](https://www.aws.training/Support#contact-us)に該当する項目がなさそうなので、[サポート](https://support.aws.amazon.com/#/contacts/aws-training)に問い合わせる。2日後に以下の内容がメールで届く。

> AWS認定特典の「無料模擬試験バウチャー」はPSIのみご利用いただけるため、お手数ですが、ピアソンVUEではなく、PSIにてご購入くださいませ。

**マジか！！！**

## PSIで申し込む

PSIで申し込んだら無事にバウチャーを適用できた。

![開始準備完了](https://storage.googleapis.com/zenn-user-upload/9451fbf45b8c0083dc92bbfe.png)

よくある質問に載っていてもよさそうだけど、同じ質問する人は少ないのかな？掲載を見落としていただけならゴメンナサイ…

# おわりに

というわけで、模擬試験はPSIで予約しましょう。
